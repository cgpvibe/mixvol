# mixvol 🚀

A high-performance creative portfolio and interactive playground built with Next.js. This platform serves as a curated space for showcasing production-ready front-end architectures and advanced web animation/interaction experiments.

---

## 📌 Project Overview

**mixvol** is designed to solve two core challenges: professional portfolio management and technical interaction journaling. Instead of static showcases, this platform focuses on live, dynamic user experiences, serving as a personal creative lab where UI/UX boundaries are pushed.

### Core Purposes:
* **Interactive Architecture Hub:** A structured repository to document, test, and stack complex web animations, micro-interactions, and visual experiences.
* **Unified Portfolio Engine:** A seamless, blazing-fast interface demonstrating modern front-end engineering, responsive design, and production-grade optimization.
* **Open Exploration:** Built with modularity in mind to serve as a high-quality codebase reference for creative developers.

---

## 🛠️ Tech Stack & Architecture

* **Framework:** Next.js (App Router, `src/` directory structure)
* **Styling:** Tailwind CSS (Utility-first, fluid responsive layouts)
* **Animation & Interaction:** GSAP (GreenSock Animation Platform) & Framer Motion for high-fidelity timeline controls and scroll-driven fluid physics.
* **Package Manager:** yarn
* **Path Aliasing:** `@/*` configured for clean, absolute import management.

---

## ✨ Key Features & Experience Pages

* **Interactive Playground:** Isolated sandboxes dedicated to specific animation challenges (e.g., advanced scroll triggers, WebGL-like canvas transitions, magnetic UI elements).
* **Performance-First Render:** Leveraging Next.js Server Components (RSC) alongside optimized client-side animation hooks to prevent layout shifts and maintain 60+ FPS.
* **Modular Code Structure:** Every interaction is built as a reusable, atomic component for rapid prototyping in future production environments.
