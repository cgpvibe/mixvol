# Antigravity 프로젝트 규칙

## Git 커밋 메시지 규칙

Git 커밋 메시지를 자동 생성할 때는 반드시 **한국어**로 작성한다.

### 커밋 메시지 형식
```
타입: 간결한 설명 (한국어)

- 변경 사항 상세 내용 (필요 시)
```

### 타입 종류
- `feat`: 새로운 기능 추가
- `fix`: 버그 수정
- `style`: 스타일/디자인 변경
- `refactor`: 코드 리팩토링
- `chore`: 설정, 의존성 등 기타 작업
- `docs`: 문서 수정
- `remove`: 파일 또는 코드 삭제

### 예시
```
feat: 메인 페이지 히어로 섹션 애니메이션 추가

- ScrollTextReveal 컴포넌트 적용
- GSAP 기반 등장 효과 구현
```
